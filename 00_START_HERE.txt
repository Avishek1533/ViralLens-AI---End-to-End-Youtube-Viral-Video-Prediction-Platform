💻 EXAMPLE COMMANDS
================================================================================

PYTHON:
  import requests
  response = requests.post("http://localhost:8000/predict", json={
      "title": "Amazing Video Title!!!",
      "model": "random_forest"
  })
  result = response.json()
  print(f"Trending: {result['is_trending']}")

CURL:
  curl -X POST http://localhost:8000/predict \
    -H "Content-Type: application/json" \
    -d '{"title":"Amazing Video!!!","model":"random_forest"}'

JAVASCRIPT:
  const video = {title: "Amazing Video!!!", model: "random_forest"};
  fetch('http://localhost:8000/predict', {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify(video)
  })
  .then(r => r.json())
  .then(d => console.log(d.is_trending));

🔧 TROUBLESHOOTING
================================================================================

❌ "Connection refused" or "Could not connect"
   → Make sure server is running
   → Run: run_server.bat (Windows) or bash run_server.sh (Linux/Mac)

❌ "Port 8000 already in use"
   → Use different port: uvicorn main:app --port 9000
   → Then use: http://localhost:9000/docs

❌ "ModuleNotFoundError: No module named..."
   → Install dependencies: pip install -r requirements.txt

❌ "Model not found"
   → Verify .joblib files are in same directory as main.py
   → Run: python verify_setup.py

❌ Slow predictions
   → Use Random Forest or Logistic Regression
   → Add workers: uvicorn main:app --workers 4

More help: See QUICKSTART.md and API_GUIDE.md

🌐 PREDICTION PARAMETERS (Optional)
================================================================================

REQUIRED:
  • title: Video title (string)

OPTIONAL:
  • description: Video description
  • tags: Tags separated by |
  • view_count: Expected views (default: 100,000)
  • likes: Expected likes (default: 5,000)
  • comment_count: Expected comments (default: 1,000)
  • category_id: YouTube category (default: 24)
  • publish_hour: Hour of publication 0-23 (default: 12)
  • publish_dow: Day of week 0-6 (default: 3)
  • model: Which model to use (default: random_forest)
