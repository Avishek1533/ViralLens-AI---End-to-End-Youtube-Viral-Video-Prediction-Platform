===============================================================================
                        API ENDPOINTS SUMMARY
===============================================================================

PUBLIC ENDPOINTS:

GET /
  Returns: API information and available models
  
GET /health
  Returns: Health status and loaded models count
  
POST /predict
  Input: Video metadata (title, description, etc.)
  Output: Trending prediction with probability
  
POST /predict-batch
  Input: Array of video metadata (up to 100)
  Output: Array of predictions
  
GET /models
  Returns: Details about all available models
  
GET /features-info
  Returns: Description of all features used
  
GET /docs
  Returns: Interactive Swagger UI
  
GET /redoc
  Returns: ReDoc documentation
